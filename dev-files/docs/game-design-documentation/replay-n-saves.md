Como a gameplay do jogo consiste numa experiência intensa e curta, o jogo não conta com salvamento durante as partidas, ainda que armazenando as estatísticas dos jogadores num placar. Se o jogador desistir será registrado sua pontuação atual. 

## Fator Replay
O Fator replay do jogo consiste no desafio de precisa dos movimentos feito pelo jogador equilibrando-se com acrobacias para subir cada vez mais a torre. Ao terminar uma vez o jogador se sentirá desafiado a subir mais rápido e fazer mais pontos para competir com os recordes anteriores.
