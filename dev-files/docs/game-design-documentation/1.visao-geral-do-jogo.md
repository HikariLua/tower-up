# Visão Geral do Jogo

1. ## Conceito
    O jogo consiste num *Infinite Climber*, focado na precisão dos pulos entre as plataformas e os reflexo ao ataques inimigos. 

    Com a progressão se desenvolvendo "para cima", o jogador sente-se motivado a subir a cada nível, se aproximado do topo, a fim de completar a fase. 

    Com dificuldade progressiva, aumentando os obstáculos e diminuindo a área para o jogador se movimentar, a tensão envolve bater recordes, conseguindo acumular pontos com agiliidade nos movimentos, precisão nos pulos e velocidade ao completar a fase.
