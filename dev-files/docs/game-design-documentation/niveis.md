As fases o jogo consistem numa progressão vertical crescente, sendo a base o nível mais básico, sendo a tela inicial do jogo, com a dificuldade da jogabilidade, número de armadilhas e frequência de inimigos, aumenta junto com o altura (em metros) que o jogador consegue alcançar. 

1. Base de Mineração
2. Catacumbas do Castelo
3. Corredores do Castelo
4. Aposentos Reais 
5. O paraiso
