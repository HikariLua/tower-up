# Físicas
* Física 3D
* Gravidade normal de 9,8
* Personagem forças como a gravidade, colisão ou empurrão de inimigos.
* Inimigos e outros objetos além do jogador sofre colisões e outras forças do ambiente.
* 
# Mecânicas
* Movimentação do jogador pelo espaço 3D
* Física de colisão entre todos os objetos
* O jogador possui inercia e pode alcançar pulos mais longes se tiver mais velocidade
* 

# Objetos
* Caixas
* Bombas
	* Bomba Grande 
	* Barril de Dinamite
	* Banana de Dinamite
# Combate
* O combate do jogo consiste unicamente em empurrões do jogador com as entidades do jogo. O jogador pode sofrer condições que afetam sua movimentação, estas são listadas a seguir:
	* com frio, o jogador perde velocidade *(speed)* 
	* em chamas, o jogador perde controle do personagem com movimentos semi-aleatorios para a direções no eixo-y.
	* golpeado, o jogador fica imobilizado por alguns segundos
